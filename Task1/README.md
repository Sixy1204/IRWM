##Usage
python main.py -h</br>  
usage: main.py [-h] [-e E] [-r] [-i I] [-o O] [-m M] [-d D]  
   
optional arguments:  
  -h, --help  show this help message and exit  
  -e E        experiment method  
  -r          turn on feedback  
  -i I        query file path  
  -o O        output ranked file .csv  
  -m M        model dir  
  -d D        docs dir
